Arduino iPod Project
- Read input from a potentiometer 
- Play one of 4 songs accordingly through a speaker using Arduino tone( ) function
- Light up a RGB LED interface to aid in song selection

Developed as an addition to the OPS Program for 2019-2020. Illustrates the difference between Digital and Analog signals,
as well as demonstrating PWM and its applications. Encourages creativity as students program their own songs.
